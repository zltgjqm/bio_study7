새로운 Biology Knowledge 추가

Paper 또는 개인 Knowledge를 입력하면 이 브라우저에 저장되고, 홈 검색과 연결 페이지에 바로 반영됩니다. 여러 Disease가 있는 Paper는 Disease별 note를 따로 저장할 수 있어요.

Paper 기반 입력

논문 하나를 읽고 발견한 Disease, Gene, Cell Type, Tissue 관계를 저장합니다.

Diseases 입력칸에 쉼표로 여러 Disease를 쓰면 note 칸이 자동으로 생깁니다.

비워도 저장 가능하지만, 논문 안에 독립된 지식이 2개 이상이면 여기에 나눠 넣으면 Entity 페이지에서 따로 모입니다.

Knowledge 기반 입력

논문이 아니어도, 실험 메모/랩미팅/리뷰 중 알게 된 Biology 지식을 저장합니다.

GPT JSON 붙여넣기

ChatGPT가 정리한 JSON 하나 또는 배열을 붙여넣고 저장하세요. Paper와 Knowledge 둘 다 가능합니다.

    Backup / Import / Export
